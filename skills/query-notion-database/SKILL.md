## 概要

Notion上の各種データベースからレコードを検索・取得する。

## 使用条件

ユーザーがデータの参照・一覧取得・検索の意図を示した場合に使用する。

## トリガー例

- 「今のタスク一覧」
- 「進行中のタスクは？」
- 「スクラップ見せて」
- 「今日のデイリープランは？」

## 対応データベース

- タスク
- デイリープラン
- スクラップ
- ナレッジ

## パラメータ

- database（必須）: 対象データベース名
  - 上記「対応データベース」から選択
- filter（任意）: 検索条件
  - ステータス（例: 未着手 / 進行中 / 完了）
  - 日付（例: 今日 / 明日 / 今週）
  - キーワード（タイトル・本文検索）
- sort（任意）: ソート条件
  - 作成日 / 更新日 / 優先度 など
- limit（任意）: 取得件数

## デフォルト動作

- filter 未指定 → 全件取得
- sort 未指定 → 作成日時の降順
- limit 未指定 → 最大件数（例: 20件）

## 出力

- レコード一覧
  - タイトル
  - ステータス（存在する場合）
  - 日付（存在する場合）
  - 概要 / 内容
  - URL（Notionリンク）
  - その他主要プロパティ
